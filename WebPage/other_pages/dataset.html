Preparing Dataset

Before training a model, the dataset of images should be normalized and resized to make it more suitable for computer vision.

First of all, we need to import all the required libraries and modules.

Write the path for the training images folder.

Resize the images, so all the pictures are the same size.

Create a data loader from the resized dataset

Calculate the mean and standard deviation. We will need this to prepare the dataset for building a neural network.

Now, we can start building a neural network by using the prepared dataset.
